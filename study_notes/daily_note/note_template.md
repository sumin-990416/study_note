```
### Calendar 플러그인 활용 (더 쉬움!)

1. **Calendar 플러그인 설치**
```
Settings → Community plugins
→ "Calendar" 검색 → Install → Enable
```

2. **Daily notes 설정**
```
Settings → Daily notes
→ Date format: YYYY-MM-DD
→ New file location: Daily (폴더 지정)
→ Template file location: Templates/Daily Note (템플릿 지정)
